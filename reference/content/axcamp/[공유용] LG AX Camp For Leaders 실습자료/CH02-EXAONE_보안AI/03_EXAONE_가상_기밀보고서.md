## 1. Executive Summary

2026년 상반기 글로벌 AI 가전 시장은 전년 대비 34.2% 성장하며 본격적인 확산기에 진입하였다. LG전자는 자체 AI 모델 EXAONE 3.5를 가전 제품군에 탑재하여 **"AI 가전 생태계"** 구축을 가속화하고, 2026 하반기 두 개의 전략 프로젝트(Project Aurora, Project Nebula)를 통해 프리미엄 시장 점유율 확대와 구독 매출 기반 확보를 동시에 추진한다.

**핵심 목표:**
- 2026 하반기 AI 가전 매출 **8.7조원** 달성 (전사 매출의 22.3%)
- 프리미엄 AI 가전 시장 글로벌 점유율 **28.5%** 확보 (현재 23.1%)
- EXAONE 기반 구독 서비스 MAU **1,200만 명** 돌파
- AI 가전 영업이익률 **9.8%** 달성 (현재 7.2%)

## 2. 신제품 로드맵

### 2.1 Project Aurora (차세대 올인원 AI 허브)

| 항목 | 상세 |
|------|------|
| **코드명** | Project Aurora |
| **제품 카테고리** | AI 홈 허브 (신규 카테고리) |
| **양산 목표** | 2026년 9월 |
| **출시 지역** | 한국, 미국, 유럽 (1차) / 동남아, 중동 (2차, 2027 Q1) |
| **예상 소비자가** | 299만원 (KR) / $1,999 (US) |
| **연간 판매 목표** | 45만대 (2026 하반기 15만대) |

**핵심 기능:**
- EXAONE 3.5 온디바이스 탑재 (NPU 18TOPS 전용칩 LG-N1 탑재)
- 가정 내 전 가전 통합 제어 (ThinQ 연동 200+ 기기)
- 음성 + 비전 + 텍스트 멀티모달 인터페이스
- 프라이버시 모드: 모든 데이터 로컬 처리 옵션
- 에너지 최적화 AI (가구 전력 사용량 평균 23% 절감 목표)

### 2.2 Project Nebula (AI 냉장고 플랫폼 3세대)

| 항목 | 상세 |
|------|------|
| **코드명** | Project Nebula |
| **제품 카테고리** | 프리미엄 AI 냉장고 |

| **출시 지역** | 글로벌 동시 출시 |
| **예상 소비자가** | 580만원 (KR) / $3,899 (US) |
| **연간 판매 목표** | 32만대 (2026 하반기 8만대) |

**핵심 기능:**
- 식재료 자동 인식 및 재고 관리 (비전 AI, 정확도 97.3%)

- 자동 식재료 주문 연동 (쿠팡, 마켓컬리, Amazon Fresh 파트너십)
- 식품 유통기한 예측 AI (음식물 쓰레기 40% 감소 목표)
- 32인치 투명 OLED 디스플레이 탑재
