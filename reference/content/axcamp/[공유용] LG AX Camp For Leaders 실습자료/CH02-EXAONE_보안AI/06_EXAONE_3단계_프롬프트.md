## 실습 개요

| 단계 | 주제 | 소요 시간 | 핵심 포인트 |
|------|------|----------|------------|
| Stage 1 | 텍스트 기반 대화 | 10분 | EXAONE의 한국어 특화 성능, 사내 AI의 보안 이점 |

## Stage 1: 텍스트 기반 대화

### 강사 노트
```
[목적]
- EXAONE 기본 대화 기능 체험
- ChatGPT/Gemini 대비 한국어 특화 성능 체감
- "왜 사내 AI인가?"에 대한 보안 맥락 이해

[진행 방법]
1. 도입 프롬프트로 EXAONE의 정체성 확인
2. LG 사업 관련 질문으로 한국어 도메인 지식 테스트
3. 외부 AI와의 차이점을 참가자가 직접 체감하도록 유도
   - 가능하면 동일 질문을 ChatGPT에도 입력하여 비교 시연

[포인트]
- EXAONE은 LG 내부 데이터로 학습되어 LG 맥락에 강함
- 한국어 비즈니스 문서 작성 품질이 높음
- 대화 내용이 외부로 유출되지 않는 보안 환경
```

### 프롬프트 1-1: 도입 (EXAONE 정체성 확인)

```
당신은 LG의 AI 모델 EXAONE입니다.
다음 질문에 답해주세요.

1. EXAONE만의 차별화된 강점은 무엇인가요?
2. ChatGPT나 Gemini와 비교했을 때,
   LG 임직원이 EXAONE을 사용해야 하는 이유는 무엇인가요?
3. 사내 기밀 문서를 다룰 때 외부 AI 대비
   EXAONE의 보안상 이점을 설명해주세요.
```

### 프롬프트 1-2: 한국어 비즈니스 능력 테스트

```
LG전자 H&A사업본부의 2026년 하반기 사업 환경을 분석해주세요.

다음 관점에서 각각 3가지씩 정리해주세요:
- 기회 요인 (Opportunities)
- 위협 요인 (Threats)

글로벌 AI 가전 시장 트렌드, 경쟁사 동향,
규제 환경 변화를 포함하여 분석하되,
임원 보고서 형식으로 작성해주세요.
```

### 프롬프트 1-3: 한국어 특화 확인 (비교 시연용)

```
다음 문장을 임원 보고서에 적합한 격식체로 다듬어주세요.
원문의 의미는 유지하되, 경영진이 읽기에 적합한
전문적이고 간결한 표현으로 수정해주세요.

원문:
"AI 가전 시장이 엄청 빨리 커지고 있어서 우리도 빨리 뭔가 해야 할 것 같습니다.
삼성이 비슷한 걸 먼저 내놓을 수도 있으니까 서두르는 게 좋겠습니다.
예산이 좀 많이 필요한데 투자 대비 효과는 괜찮을 것 같습니다."
```

## Stage 2: 파일 첨부 (컨텍스트 덤핑)

### 강사 노트
```
[목적]
- 기밀 문서를 AI에 첨부하여 분석하는 "컨텍스트 덤핑" 기법 체험
- 외부 AI(ChatGPT, Gemini)에는 절대 올릴 수 없는 문서도
  EXAONE에서는 안전하게 활용할 수 있음을 체감
- 문서 기반 질의응답(RAG 유사 경험)의 실용성 확인

[진행 방법]
1. 03_EXAONE_가상_기밀보고서.md 파일을 EXAONE에 첨부
2. 아래 프롬프트를 순서대로 입력
3. AI가 문서 내용을 정확히 이해하고 분석하는지 확인

[포인트]
- "이 문서를 ChatGPT에 올릴 수 있나요?" → 절대 안 됨 (사내한/대외비)
- "EXAONE이라면?" → 사내 AI이므로 안전하게 분석 가능
- 이것이 사내 AI의 가장 큰 실무적 가치
- 파일 첨부 = "컨텍스트 덤핑"의 가장 기본적인 형태

[주의사항]
- 가상 문서임을 반드시 안내 (실제 LG 전략과 무관)
- 실무에서는 실제 사내 보고서, 재무 데이터, 전략 문서 등을
  동일한 방식으로 활용 가능
```

### 프롬프트 2-1: 문서 요약 및 핵심 파악

```
첨부한 문서는 LG전자의 2026 하반기 AI 가전 사업 전략 보고서입니다.

이 보고서를 경영진 브리핑용으로 요약해주세요.
다음 항목을 포함하여 A4 1장 이내로 정리해주세요:

1. 핵심 전략 방향 (3줄 요약)
2. 주요 신제품과 출시 일정
3. 매출 목표 및 핵심 KPI
4. 가장 큰 리스크 3가지와 대응 방향
```

### 프롬프트 2-2: 심층 분석 및 비판적 검토

```
첨부한 전략 보고서에 대해 "사업부 전략검토회의"에서
나올 수 있는 날카로운 질문 5가지를 만들어주세요.

각 질문에 대해:
- 예상 질문
- 질문의 배경 (왜 이 질문이 중요한지)
- 보고서 내용 기반 답변 초안

형식으로 정리해주세요.

특히 원가 구조, M&A 리스크, 경쟁사 대응 측면에서
비판적으로 검토해주세요.
```

### 프롬프트 2-3: 데이터 기반 의사결정 지원

```
첨부 보고서의 재무 데이터를 분석하여 다음을 답해주세요:

1. Project Aurora와 Project Nebula 중
   투자 우선순위를 정한다면 어느 쪽인가요?
   매출총이익률, 영업이익률, 시장 규모를 기준으로 비교 분석해주세요.

2. EXAONE 구독 서비스의 매출총이익률이 78.8%입니다.
   하드웨어 대비 구독 모델의 장기적 가치를 분석하고,
   하드웨어와 구독의 최적 매출 비율을 제안해주세요.

3. M&A 검토 대상 4개 기업 중
   ROI 관점에서 우선순위를 매기고 근거를 제시해주세요.

표 형식으로 정리해주세요.
```

## Stage 3: 딥 리서치

### 강사 노트
```
[목적]
- EXAONE의 심층 리서치(Deep Research) 기능 체험
- 단순 질의응답을 넘어, 복합적 분석과 전략 제안 능력 확인
- 사내 보안 환경에서 외부 시장 정보와 내부 전략을 결합하는 활용법

[진행 방법]
1. EXAONE 딥 리서치 모드 활성화 (해당 기능이 있는 경우)
2. 아래 프롬프트 입력
3. AI가 여러 소스를 종합하여 심층 분석하는 과정 관찰

[포인트]
- 딥 리서치 = AI가 스스로 여러 단계로 조사하고 종합하는 기능
- 임원의 "비서"처럼 복잡한 조사를 대신 수행
- 사내 AI이므로 "우리 회사 전략 맥락"을 포함한 질문이 가능
- 외부 AI에서는 "기밀 유출 우려"로 할 수 없는 질문 유형

[주의사항]
- 딥 리서치 기능은 응답 시간이 길 수 있음 (2~5분)
- 결과의 깊이와 구조화 정도에 주목하도록 안내
- 딥 리서치가 없는 환경이면 일반 프롬프트로 대체 가능
```

### 프롬프트 3-1: 시장 전략 심층 리서치

```
LG전자가 2026 하반기에 "AI 홈 허브" 카테고리(Project Aurora)로
신규 진출하려 합니다.

다음 관점에서 심층 리서치를 수행해주세요:

1. 글로벌 AI 홈 허브 시장 현황
   - 시장 규모, 성장률, 주요 플레이어
   - Amazon Echo, Google Nest, Apple HomePod 대비 포지셔닝 전략

2. LG의 차별화 전략 제안
   - EXAONE 온디바이스 AI + 프라이버시가 경쟁 무기가 될 수 있는
     시장 세그먼트는?
   - "가전 연동 247종"이라는 강점을 극대화할 Go-to-Market 전략은?

3. 가격 전략
   - 299만원($1,999)이 적정 가격인지 시장 분석 기반으로 평가
   - 구독 번들(Aurora + EXAONE Premium) 전략의 실현 가능성

보고서 형식으로 작성해주세요.
```

### 프롬프트 3-2: 경쟁 대응 시나리오 분석

```
다음 시나리오에 대해 심층 분석해주세요:

시나리오: 삼성전자가 2026년 10월에
"Bixby AI Hub"를 LG Aurora 대비 30% 저렴한 가격으로 출시하고,
Galaxy 스마트폰 사용자에게 무료 1년 구독을 제공한다면?

분석 항목:
1. LG에 미치는 영향 (매출, 시장점유율 시뮬레이션)
2. 즉각 대응 방안 (3개월 내)
3. 중장기 대응 전략 (6~12개월)
4. 이 시나리오가 실현될 확률과 그 근거
5. LG가 선제적으로 취해야 할 방어 조치

경영전략회의에서 발표할 수 있는 수준으로 작성해주세요.
워스트 케이스, 베스트 케이스, 기본 케이스 3가지 시나리오를 포함해주세요.
```

### 프롬프트 3-3: 사내 AI 활용 전략 (메타 질문)

```
LG전자 임원의 입장에서, EXAONE을 일상 업무에
가장 효과적으로 활용할 수 있는 방법을 리서치해주세요.

다음을 포함해주세요:

1. 임원급이 EXAONE을 활용하면 생산성이 높아지는
   Top 5 업무 시나리오 (구체적 프롬프트 예시 포함)

2. 보안 등급별 AI 활용 가이드라인 제안
   - 사내한 문서: EXAONE에서 가능한 작업 범위
   - 대외비 문서: 추가 주의사항
   - 극비 문서: AI 활용 제한 기준

3. "AI를 잘 쓰는 임원"과 "AI를 못 쓰는 임원"의 차이
   - 프롬프트 작성 습관의 차이
   - 업무 프로세스에 AI를 통합하는 방법

4. 3개월 후 측정할 수 있는 AI 활용 KPI 제안

실무에 바로 적용할 수 있도록 구체적으로 작성해주세요.
```
