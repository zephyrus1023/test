# CH07 Antigravity 실습: HTML + PPT 동시 생성 프롬프트

> **실습 도구**: Antigravity (https://antigravity.so)
> **목표**: 하나의 프롬프트로 인터랙티브 HTML 대시보드 + 이사회 보고용 PPT를 동시에 생성
> **소요 시간**: 약 15~20분

---

## 강사 노트

```
[진행 가이드]
- 이 프롬프트는 Antigravity에 그대로 복사-붙여넣기 가능
- 수강생에게 분석 대상 데이터를 먼저 업로드하도록 안내
  (예: 엑셀 매출 데이터, CSV 실적 자료 등)
- 데이터가 없는 수강생용으로 샘플 데이터 준비 필요
- LG 브랜드 컬러를 적용한 산출물이 나오는 것이 핵심 포인트
- "같은 분석, 두 가지 포맷" → 보고 대상에 맞는 산출물 자동 생성
```

## LG 브랜드 가이드라인 참고

| 항목 | 값 | 용도 |
|------|-----|------|
| Heritage Red | `#A50034` | 메인 강조색, 제목, KPI 하이라이트 |
| Dark Gray | `#333333` | 본문 텍스트 |
| Light Gray | `#F5F5F5` | 배경, 카드 배경 |
| White | `#FFFFFF` | 기본 배경 |
| 보조색 1 | `#6B7280` | 부제목, 보조 텍스트 |
| 보조색 2 | `#E5E7EB` | 구분선, 테두리 |
| 서체 (PPT) | LG Smart (없으면 Pretendard 또는 맑은 고딕) |
| 서체 (HTML) | Pretendard, -apple-system, sans-serif |

## 프롬프트 (복사-붙여넣기용)

```
업로드한 데이터를 분석하여 아래 2가지 산출물을 동시에 생성해주세요.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📊 산출물 1: 인터랙티브 HTML 대시보드
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

## 대시보드 구성

### 상단 영역: KPI 카드 (4개)
- 총 매출 (전년 대비 증감률 표시)
- 영업이익률
- 주요 성장 부문 (Top 1)
- 전략 과제 달성률
- 카드 디자인: 흰색 배경, 그림자 효과, 숫자는 Heritage Red(#A50034) 강조

### 중단 영역: 차트 섹션
- 차트 1: 월별 매출 추이 (Line Chart) - 최근 12개월
- 차트 2: 부문별 매출 비중 (Donut Chart)
- 차트 3: 전년 동기 대비 성장률 (Bar Chart)
- 차트 라이브러리: Chart.js 사용
- 차트 컬러 팔레트: #A50034(메인), #333333, #6B7280, #E5E7EB 순서

### 하단 영역: 상세 데이터 테이블
- 정렬 가능한 데이터 테이블
- 컬럼: 부문 | 매출 | 영업이익 | 성장률 | 전략과제
- 행 클릭 시 상세 정보 표시
- 검색/필터 기능

### 인터랙티브 기능
- 기간 필터: 분기별 / 반기별 / 연간 전환
- 부문 필터: 체크박스로 특정 부문 선택/해제
- 차트 호버 시 상세 데이터 툴팁
- 다크/라이트 모드 토글
- "PDF 내보내기" 버튼 (window.print 활용)

### HTML 기술 사양
- 단일 HTML 파일 (CSS/JS 인라인)
- Chart.js CDN: https://cdn.jsdelivr.net/npm/chart.js
- 폰트: Pretendard (CDN) 또는 시스템 폰트 폴백
- 반응형: 모바일 대응 (min-width 기반 미디어쿼리)
- 인코딩: UTF-8

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📑 산출물 2: 이사회 보고용 PowerPoint
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

## PPT 구성 (5~7 슬라이드)

### 슬라이드 1: 표지
- 제목: "[부문명] 경영 실적 보고"
- 부제: 보고 기간, 작성일
- 하단: LG 로고 위치 (placeholder 텍스트로 "LG" 표시)
- 배경: 흰색, 좌측에 Heritage Red(#A50034) 세로 액센트 바

### 슬라이드 2: Executive Summary
- 핵심 성과 3개 (불릿 포인트)
- 주요 리스크 2개
- 다음 분기 전망 한 줄
- 레이아웃: 좌측 텍스트 + 우측 KPI 박스

### 슬라이드 3: 매출 실적
- 월별/분기별 매출 추이 차트
- 전년 동기 대비 증감 표시
- 주요 변동 요인 2~3개 불릿

### 슬라이드 4: 부문별 분석
- 부문별 매출 비중 차트 (파이/도넛)
- 각 부문 한줄 코멘트
- 성장/하락 부문 하이라이트

### 슬라이드 5: 전략 과제 진행 현황
- 과제별 달성률 바 차트
- 상태 표시: 정상/주의/지연
- 핵심 이슈 및 대응 방안

### 슬라이드 6: 리스크 & 기회 요인 (선택)
- 2x2 매트릭스: 리스크 vs 기회 / 단기 vs 중장기
- 각 항목별 한줄 설명

### 슬라이드 7: Next Steps (선택)
- 다음 분기 핵심 실행 과제 3~5개
- 담당 부문 / 기한 명시
- "의사결정 요청 사항" 별도 박스

## PPT 디자인 사양
- 슬라이드 크기: 16:9 (와이드스크린)
- 메인 컬러: Heritage Red #A50034
- 텍스트 컬러: #333333 (본문), #FFFFFF (강조 배경 위)
- 배경: 흰색 기본, 포인트 슬라이드만 Heritage Red 배경
- 제목 폰트: 굵게, 24pt 이상
- 본문 폰트: 14~16pt
- 차트/그래프 색상: #A50034, #333333, #6B7280, #E5E7EB 팔레트
- 여백: 상하좌우 충분히 (임원 보고용이므로 텍스트 밀집 금지)
- 슬라이드당 핵심 메시지 1개 원칙

🎯 공통 지시사항
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 데이터가 부족한 부분은 합리적 가정으로 채워주세요
2. 모든 숫자에 천 단위 구분 쉼표 적용
3. 금액 단위: 억 원
4. 증감률은 소수점 1자리까지 (예: +12.3%)
5. 차트와 표의 데이터는 HTML과 PPT에서 동일하게 유지
6. 한국어로 모든 텍스트 작성
```

## 프롬프트 커스터마이징 가이드

```
[강사 안내]
수강생이 자신의 업무에 맞게 프롬프트를 수정하도록 안내하세요.

■ 변경 가능 항목:
  - KPI 카드 항목 (매출 → 고객만족도, 생산량 등)
  - 차트 종류 (Line → Area, Bar → Waterfall 등)
  - PPT 슬라이드 구성 (부문별 분석 → 지역별 분석 등)
  - 브랜드 컬러 (자사 브랜드 컬러로 변경)

■ 예시 변형:
  - 마케팅 부문: 캠페인 성과 대시보드 + 월간 마케팅 보고 PPT
  - HR 부문: 인력 현황 대시보드 + 이사회 인사 보고 PPT
  - R&D 부문: 특허/기술 동향 대시보드 + 기술 전략 보고 PPT
```
