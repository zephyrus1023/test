# Gemini 소개 및 접속 방법

오늘 실습의 기본 workbench는 Gemini다. 질문으로 시작하고, 파일을 붙이고, 같은 대화에서 분석을 이어가다가, 마지막에는 Gems로 나만의 AI 비서까지 만듭니다.

## 오늘 Gemini에서 경험할 것

1. Chat UI

질문, 요약, 표 정리, 초안 작성처럼 짧은 왕복이 많은 업무를 Gemini에서 바로 처리합니다.

2. Multimodal

텍스트만이 아니라 PDF, 오디오, 회의 맥락 자료를 붙여 같은 대화에서 분석을 심화합니다.

3. Reusable Assistant

반복되는 프롬프트는 Gems로 저장해, 회의 분석과 브리핑 작성 업무를 개인 비서 형태로 재사용합니다.

### 오늘 수업 연결

이 챕터는 **Gemini 접속과 인터페이스 이해 → 구조화된 프롬프트 실습 → Gems로 역할 저장** 순서로 진행합니다. 먼저 익숙해지고, 그 다음 잘 시키고, 마지막에 반복 업무를 고정하는 흐름입니다.

## 접속 후 바로 확인할 3가지

- **[gemini.google.com/app](https://gemini.google.com/app)**에 접속해 Google 계정으로 로그인합니다.

- 하단 입력창 우측의 **모드 선택**에서 오늘 실습에 맞는 모델을 확인합니다. 기본은 빠른 모델이지만, 긴 문서 분석과 회의 실습은 상위 모델이 더 유리합니다.

- 입력창 좌측의 **+** 버튼으로 파일 업로드 메뉴를 열어 오디오, 문서, 이미지 등 실습 자료를 붙입니다.

## Gemini 화면 간단 소개

세 장 모두 클릭하면 크게 볼 수 있고, 확대 모달에서 좌우로 넘기며 홈 화면, 모드 선택, 파일 업로드 메뉴를 순서대로 확인할 수 있습니다.

## 오늘 주로 쓰는 기능

| 기능 | 어떻게 쓰나 | 오늘 실습 연결 |
| --- | --- | --- |
| **Chat** | 질문, 요약, 표 정리, 초안 작성 | 비즈니스 프롬프트 4종 실습 |
| **파일 첨부** | 오디오, 보고서, 맥락 문서 추가 | 회의 녹음 분석과 교차 검증 |
| **멀티턴 대화** | 같은 대화에서 후속 질문을 이어가기 | 회의록 → 이슈 추출 → 전략 제안 |
| **Deep Research** | 넓은 범위의 조사와 보고서 작성 | 후속 리서치 확장 옵션 |
| **Gems** | 반복 업무용 맞춤형 AI 비서 저장 | 경영진 브리핑 비서 만들기 |
