CH 05 플랫폼

NotebookLM 소개 및 문서 기반 AI 연구 도우미

NotebookLM은 내가 넣은 자료를 중심으로 답하고 모든 응답에 출처를 다는 grounded research 도구다. CH03에서 만든 회의록·Deep Research 메모·구조화된 리서치 결과를 여기에 넣으면, 내부 자료와 외부 자료를 함께 비교하며 더 신뢰도 높은 분석으로 확장할 수 있다.
CH03에서 가져오는 입력
앞 단계에서 만든 것NotebookLM에 넣으면 좋은 이유NotebookLM에서 기대하는 역할
회의록과 핵심 이슈 정리내부 문제의식과 의사결정 포인트가 선명해진다기존 문서와 대조하며 빠진 근거를 찾음
ChatGPT Deep Research 메모외부 조사 결과를 보조 소스로 추가할 수 있다내부 문서와 외부 리서치를 함께 비교 분석
구조화된 Markdown 메모제목·불릿이 분명해서 NotebookLM이 다루기 쉽다스튜디오 산출물과 후속 질문의 바탕 자료가 됨
연결 포인트
CH03에서 만든 결과는 끝이 아니라, NotebookLM에 넣을 수 있는 추가 소스입니다. 여기서부터는 “잘 쓴 프롬프트”보다 “좋은 소스를 함께 묶는 것”이 더 중요해집니다.
언제 NotebookLM을 써야 하나
1. Grounded Research
PDF, Google Docs, 웹 URL, 이미지, 텍스트 등 최대 50개 소스를 함께 읽고 소스 기반으로만 답을 만듭니다.
2. Citation First
응답마다 출처 번호가 붙어서, 어떤 문서의 어떤 부분을 근거로 말하는지 바로 추적할 수 있습니다.
3. Enterprise Lens
사내 자료, 시장 보고서, 외부 뉴스, 브랜드 자산을 묶어 조직만의 해석과 시사점을 만들기 좋습니다.
4. Studio Output
보고서, 슬라이드, 오디오, 마인드맵처럼 공유 가능한 산출물을 같은 노트북에서 바로 만듭니다.
NotebookLM이 특히 빛나는 상황
출처가 있는 분석, 내부 문서 기반 검토, 멀티소스 교차 분석, 임원 보고 초안이 동시에 필요할 때 Gemini 단독 채팅보다 NotebookLM이 더 안정적인 선택입니다.
실제 NotebookLM 작업 화면으로 보기
왼쪽은 소스 목록, 가운데는 출처가 달린 채팅, 오른쪽은 Studio 패널입니다. NotebookLM은 읽기와 토론과 산출물 생성을 한 화면 안에서 이어가게 만듭니다.
소스 패널
업로드한 문서와 웹 자료를 한곳에 모아 관리합니다. 어떤 자료가 분석에 들어갔는지가 명확하다.
채팅 패널
출처 번호와 함께 답을 받아보고, 같은 문맥으로 후속 질문을 이어가며 깊게 파고든다.
Studio 패널
슬라이드, 오디오, 마인드맵 같은 정리된 출력물을 만듭니다. 채팅과는 역할이 다르다.
Studio 패널과 채팅은 어떻게 다르나
구분Studio 패널채팅 패널
목적 공유 가능한 산출물을 정리된 형태로 생성 질문, 검증, 후속 탐색을 이어가는 대화
대표 결과 Slide Deck, Audio Overview, Mind Map, Report 출처가 붙은 답변, 비교표, 후속 질의응답
추천 상황 보고 자료, 요약본, 교육용 산출물이 필요할 때 왜 그런 결론이 나왔는지 더 캐묻고 검증할 때
실무 감각으로 기억할 한 줄
Studio는 "정리된 결과물을 뽑는 곳", 채팅은 "그 결과를 파고들며 검증하는 곳"입니다. 둘을 같이 써야 NotebookLM의 장점이 온전히 나옵니다.
바로 시작하기
  1. notebooklm.google.com에 접속해 새 노트북을 만듭니다.
  2. 문서, 웹 URL, 이미지, 텍스트 등 오늘 분석에 필요한 소스를 먼저 모은다.
  3. 핵심 질문으로 출처 기반 답을 만든 뒤, Studio에서 슬라이드나 보고서로 정리합니다.
  4. 산출물을 본 다음 다시 채팅으로 들어가 약한 부분을 보완합니다.
