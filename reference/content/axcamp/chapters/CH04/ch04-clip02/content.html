문서 기반 AI 리서치: CIQO와 LG 스타일 브리핑

NotebookLM의 핵심은 좋은 소스를 넣고, 출처가 붙은 답을 뽑아내고, 그 결과를 다시 산출물로 전환하는 것입니다. 이번 실습에서는 CH03에서 만든 회의 맥락 문서와 Deep Research 메모를 보조 소스로 가져오고, WEF와 Deloitte 보고서, Fast Research, Studio 패널, 그리고 JSON 슬라이드 템플릿까지 연결해 더 다층적이고 전문적인 LG 스타일 브리핑으로 완성합니다.

Deep Research 메모

Markdown 구조 메모
제목과 불릿으로 정리된 메모는 NotebookLM이 다루기 좋습니다. 나중에 Studio 슬라이드와 오디오 오버뷰의 바탕 자료로도 쓰기 쉽습니다.
실습 연결
즉, CH03에서 만든 결과를 단순히 버리는 것이 아니라, NotebookLM에 추가 소스로 붙여서 맥락을 더 두텁게 만드는 흐름으로 이해하면 됩니다.

2. 소스 추가 안내
소요 시간 약 3분
내부 결과물은 텍스트로 붙여 넣고, 외부 보고서는 위 표의 Google Drive 링크를 기준으로 확보해 NotebookLM에 순서대로 추가하는 단계입니다.
  1. NotebookLM (notebooklm.google.com)에 접속합니다.
  2. 새 노트북을 클릭합니다.
  3. 노트북 이름을 AX Camp 전략 리서치로 설정합니다.
  4. 먼저 회의 맥락 문서와 앞선 실습 메모를 텍스트 소스로 붙입니다.
  5. 위 표의 링크복사 버튼으로 Google Drive 링크를 복사하고, 자료를 열어 순서대로 NotebookLM 소스로 추가합니다.
  6. WEF → Deloitte 2023 → Deloitte 2026 순으로 넣으면 인력·시장·미래 시나리오가 자연스럽게 겹쳐집니다.
  7. LG 로고는 분석용 소스가 아니라 슬라이드/인포그래픽 스타일 참고자료로 마지막에만 활용합니다.
CIQO 포인트
NotebookLM은 소스 품질과 조합에 따라 답변 수준이 달라집니다. 내부 회의 맥락 없이 WEF와 Deloitte만 읽으면 일반론으로 흐르기 쉽고, 반대로 외부 보고서 없이 회의 맥락만 읽으면 시장 기준이 부족해집니다.
3. 기본 질문 프롬프트: 다중 소스 비교 분석
소요 시간 약 5분
여기서는 회의 결과와 Deep Research 메모를 출발점으로 삼고, WEF와 Deloitte를 통해 내부 이슈를 외부 데이터와 시장 맥락으로 다시 읽어봅니다. 아래 두 질문이 특히 시작점으로 적절합니다.
프롬프트 2-1: 인력 병목 교차 검증
회의 맥락 문서에서 드러난 'AI/ML 인력 충원 병목' 이슈를 WEF Future of Jobs 2025 보고서 기준으로 다시 읽어줘. 특히 아래를 정리해 줘. - AI 시대에 수요가 빠르게 증가하는 직무와 스킬 - 왜 기업들이 AI 인력 확보에 병목을 겪는지 - LG그룹 같은 대기업이 인력 재배치와 리스킬링으로 대응할 때의 시사점 출력은 `핵심 데이터 / 해석 / 임원 액션 아이템` 3개 섹션으로 정리해 줘.
프롬프트 2-2: 시장 환경과 AX 투자 맥락 비교
회의 맥락 문서의 '하반기 AX 투자 확대' 안건과 Deloitte Luxury 2023/2026 보고서의 디지털 전환 흐름을 함께 비교해 줘. 아래 관점으로 정리해 줘. - 두 보고서가 공통으로 보여주는 시장 변화 신호 - 내부 회의의 투자 문제의식과 외부 산업 보고서가 만나는 지점 - 보고서만 봤을 때 놓치기 쉬운 내부 맥락, 내부 회의만 봤을 때 놓치기 쉬운 외부 맥락 출력은 `공통 신호 / 차이점 / LG 시사점`으로 정리해 줘.
왜 이 두 질문이 먼저 좋은가
첫 번째 질문은 인력 병목을 WEF의 데이터로 읽게 해 주고, 두 번째 질문은 AX 투자 안건을 Deloitte의 시장 변화와 연결하게 해 줍니다. 즉 내부 회의의 문제를 외부 자료로 grounded하게 다시 읽는 가장 자연스러운 출발점입니다.
중간 결과도 다시 소스가 됩니다.
위 두 질문의 답을 NotebookLM 노트로 저장해 두면, 5단계에서 다시 소스로 전환해 후속 브리핑과 Studio 산출물의 근거 자료로 재사용할 수 있습니다.
4. 패스트 리서치 프롬프트
소요 시간 약 4분
NotebookLM의 Fast Research 기능으로 웹 소스를 자동 수집하는 흐름입니다.
  1. 좌측 소스 패널에서 소스 추가를 클릭합니다.
  2. 패스트 리서치(Fast Research) 또는 웹 검색을 선택합니다.
  3. 아래 주제를 입력합니다.
  4. NotebookLM이 관련 웹 소스를 자동으로 수집해 노트북에 추가하는지 확인합니다.
패스트 리서치 주제
2025-2026년 글로벌 AI 가전 시장 트렌드와 주요 기업(LG, 삼성, 하이센스)의 AI 전략 비교 분석
대안 주제
한국 대기업의 AI 인재 양성 전략과 리스킬링 프로그램 최신 동향
5. 노트를 소스로 전환
  1. 좌측 노트 패널에서 저장된 노트를 선택합니다.
  2. 노트 상단 또는 우클릭 메뉴에서 소스로 전환을 클릭합니다.
  3. 해당 노트가 소스 목록에 추가된 것을 확인합니다.
전환 후 활용 프롬프트
방금 소스로 전환된 '인력 병목 교차 검증' 노트 또는 '시장 환경과 AX 투자 맥락 비교' 노트와 원래 WEF/딜로이트 보고서를 함께 참조해서, 각 시사점별 실행 계획을 구체화해 줘. 특히 LG그룹 임원이 다음 분기에 바로 착수할 수 있는 액션 아이템 중심으로 정리해 줘.
6. 스튜디오 패널 활용 프롬프트
소요 시간 약 4분
NotebookLM 스튜디오에서는 같은 소스를 바탕으로도 다른 형식의 산출물을 만들 수 있습니다. 아래 프롬프트는 슬라이드, 인포그래픽, 보고서, 오디오/동영상 내레이션 초안처럼 용도별 결과물을 더 재미있게 뽑아보기 위한 예시입니다.
  1. 우측 스튜디오 패널을 엽니다.
  2. 생성할 콘텐츠 유형을 선택합니다.
6-A: 슬라이드 생성 프롬프트
LG그룹 C-Level 임원 보고용 슬라이드 초안을 만들어 줘. 대상 청중: LG그룹 C-Level 임원 슬라이드 수: 8~10장 톤: 간결하고 데이터 중심, 경영진 보고 형식 스타일: 화이트 기반, 회색 중심 타이포, 절제된 LG red 포인트 참고자료: 소스에 포함된 LG 로고 파일을 시각 톤 참고자료로 활용해 줘. 포함 내용: - AX 전략 회의의 핵심 이슈 3개 - WEF 보고서의 인력·스킬 시사점 - Deloitte 보고서의 시장 전환 신호 - 다음 분기 임원 액션 아이템
6-B: 인포그래픽 / 원페이지 요약 프롬프트
한 장짜리 인포그래픽 또는 executive one-pager 초안을 만들어 줘. 목적: 바쁜 임원이 30초 안에 핵심을 파악하게 하기 톤: 숫자와 키워드가 눈에 잘 들어오는 간결한 브리핑 스타일: LG 로고를 참고해 차분한 화이트/그레이 기반, 핵심만 LG red로 강조 반드시 포함할 것: - 현재 상황을 보여주는 핵심 데이터 3개 - 내부 회의의 주요 판단 포인트 3개 - 리스크와 기회 2x2 요약 - 마지막에 'So What / Next Action' 한 줄
6-C: 보고서 생성 프롬프트
LG그룹 임원용 전략 브리핑 보고서를 작성해 줘. 분량: A4 기준 2~3페이지 톤: 단정하고 실무적인 executive briefing 구성: - 배경과 현재 상황 - 핵심 이슈 3개 - 외부 보고서 기반 교차 검증 - 임원 판단 포인트 - 다음 분기 액션 아이템 조건: - WEF와 Deloitte의 근거를 자연스럽게 인용할 것 - 내부 회의 맥락과 연결되는 의미를 반드시 써 줄 것 - 마지막에는 CEO에게 보고하는 한 문단 요약을 붙일 것
6-D: 오디오 / 동영상 내레이션 초안 프롬프트
두 명의 전문가가 AI 시대의 기업 전략을 논의하는 오디오 오버뷰 또는 동영상 내레이션 초안을 만들어 줘. 길이: 90초~2분 형식: 대화형 또는 발표형 중 자연스러운 방식 톤: 전문적이지만 너무 강의처럼 들리지 않게 강조할 것: - 내부 회의에서 드러난 핵심 문제의식 - WEF와 Deloitte가 보여주는 외부 변화 신호 - LG그룹 같은 대기업이 지금 무엇을 준비해야 하는지 핵심 데이터는 자연스럽게 대화에 녹여 설명하고, 마지막에는 '그래서 지금 무엇을 해야 하는가'로 마무리해 줘.
