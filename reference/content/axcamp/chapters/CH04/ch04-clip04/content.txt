실습 시트
실습용 파일이 필요하신가요?
참고자료 라이브러리 › 실습용 컨텍스트 파일
NotebookLM 실습을 체계적으로 진행하기 위한 단계별 가이드입니다. 초급(기본 클릭)과 상급(커스터마이징) 경로를 안내합니다.
이 실습에서는
초급 경로: 소스 업로드 → 추천 질문 클릭 → 오디오 오버뷰 기본 생성 (약 7분)
상급 경로: 패스트 리서치 → 교차 분석 → 노트→소스 전환 → 스튜디오 커스터마이징 (약 17분)
수준별 맞춤 실습으로 NotebookLM의 핵심 가치를 직접 체험합니다
초급 코스 (A Course) — 기본 클릭 중심
단계	활동	조작	소요
Step 1	소스 업로드	NotebookLM 접속 → 새 노트북 → 보고서 파일 드래그앤드롭	2분
Step 2	오디오 오버뷰 생성 시작	스튜디오 → 오디오 오버뷰 → 기본 설정으로 생성 클릭 (생성 중 대기)	1분
Step 3	대기 시간 활용: 추천 질문	오디오 생성 대기 중 → 추천 질문 클릭하여 문서 탐색	2분
Step 4	노트 저장 + 오디오 확인	유용한 답변 → "노트에 저장" 클릭 → 완성된 오디오 오버뷰 청취	2분

초급 합계: 약 7분 — "버튼만 누르면 됩니다" | 느린 작업(오디오)을 먼저 시작하고, 대기 시간에 채팅 탐색을 합니다

상급 코스 (B Course) — 커스터마이징 중심
단계	활동	조작	소요
Step 1	소스 업로드 + 패스트 리서치	파일 업로드 + Fast Research로 웹 소스 자동 수집	3분
Step 2	슬라이드·팟캐스트 생성 시작	스튜디오 → 슬라이드 커스터마이징 생성 클릭 → 오디오 오버뷰 커스터마이징 생성 클릭 (동시 트리거)	2분
Step 3	대기 시간 활용: 교차 분석	생성 대기 중 → 직접 작성한 프롬프트로 WEF·딜로이트 보고서 교차 질의	4분
Step 4	대기 시간 활용: 노트 → 소스 전환	분석 결과를 노트 저장 → 소스로 전환 → 재분석	4분
Step 5	Studio 결과물 확인	생성 완료된 슬라이드·팟캐스트 확인 및 리뷰	4분

상급 합계: 약 17분 — "AI의 핵심 가치는 커스터마이징에서 나옵니다" | 느린 작업(슬라이드·팟캐스트)을 먼저 트리거하고, 대기 시간에 채팅 분석을 합니다

Studio 패널 체험

NotebookLM Studio에서 생성할 수 있는 형태:

Audio Overview
두 명의 AI 호스트가 대화 형식으로 보고서 내용을 설명
Mind Map
보고서의 핵심 구조를 시각적 마인드맵으로 자동 생성
Slide Deck
보고서를 5~10장 슬라이드 프레젠테이션으로 자동 변환
Video Overview
문서 내용을 AI 영상으로 자동 제작
Data Tables
소스 데이터를 구조화된 표로 자동 변환, Google Sheets로 내보내기
Reports
브리핑 문서, FAQ, 타임라인 등 목적별 보고서 자동 생성
Studio 패널 활용 팁
Studio 패널 상단의 타일(Audio, Video, Mind Map, Slide Deck, Data Tables, Reports) 중 원하는 형태를 선택하세요. 같은 노트북에서 여러 종류의 출력물을 동시에 생성할 수 있으며, Audio Overview를 들으면서 Mind Map을 탐색하는 멀티태스킹도 가능합니다.
