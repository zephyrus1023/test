외부 API 연동
Build로 만든 앱에 외부 데이터 소스를 연결하여 실시간 정보를 다루는 두 가지 경로를 안내합니다.
이 실습에서는
• 초급(Google Search 그라운딩)과 상급(Firecrawl API) 두 경로의 차이를 이해합니다
• 각 경로의 장단점과 적합한 활용 상황을 파악합니다
• 본인 수준에 맞는 실습 경로를 선택합니다
초급 vs 상급 경로 비교
	초급: Google Search 그라운딩	상급: Firecrawl API 연동
사전 준비	없음 (AI Studio 내장)	Firecrawl API 키 필요
검색 범위	Google 검색 결과 기반	특정 웹페이지 직접 크롤링
데이터 깊이	뉴스 헤드라인·요약 수준	페이지 전체 콘텐츠 추출
활용 예시	경쟁사 뉴스 모니터링	IR 페이지, 기술 블로그 정밀 분석
소요 시간	~15분	~25분
주요 연동 가능 API
Google Search API: Gemini의 구글 검색 기능을 앱에 직접 연결 (초급 경로)
Firecrawl: 웹페이지를 AI가 읽을 수 있는 마크다운으로 변환 (상급 경로)
SerpAPI: 검색 엔진 결과를 구조화된 데이터로 제공
News API: 실시간 뉴스 데이터를 수집하여 분석
초급 / 상급 분기
• 초급반: 위 메인 프롬프트만 사용 → Google Search 그라운딩으로 기본 리서치 앱 완성 (~15분)
• 상급반: 메인 프롬프트 + Firecrawl API 연동 → 직접 크롤링 기능 추가 (~25분)
• 두 버전 모두 코딩 없이 프롬프트만으로 완성됩니다
