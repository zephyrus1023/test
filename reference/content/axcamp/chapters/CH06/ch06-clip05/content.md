# 실습 시트: 앱 만들기

실습용 파일이 필요하신가요? [참고자료 라이브러리 › 실습용 컨텍스트 파일](#ch08-clip04)

Google AI Studio Build로 경쟁사 리서치 대시보드를 **4단계에 걸쳐 점진적으로 완성**하는 핵심 실습입니다.

이 실습에서는

• Phase 1: AI에게 말로 지시하여 기본 리서치 대시보드를 만듭니다 (20분)  
• Phase 2: 대화로 기능을 하나씩 추가하며 앱을 고도화합니다 (20분)  
• -- 이 시점에서 10분 휴식을 가집니다 --  
• Phase 3: LG 브랜드에 맞는 전문적인 UI/UX로 다듬습니다 (20분)  
• Phase 4: \[선택\] 외부 API**API**  
프로그램끼리 대화하는 통로예요. 마치 레스토랑의 웨이터처럼, 주문(요청)을 주방(서비스)에 전달하고 요리(결과)를 가져다줍니다. 연동으로 상급 기능을 확장합니다 (15분)

Phase 1 — 기본 앱 빌드 (20분)

Phase 1 메인 프롬프트복사

경쟁사의 최신 뉴스와 기술 동향을 자동으로 수집·요약하는 리서치 대시보드를 만들어줘. 기능 요구사항: 1. 검색어 입력 → Google Search로 관련 뉴스 자동 검색 2. 각 뉴스의 핵심 내용을 3줄로 자동 요약 3. 카테고리별 필터 (기술/비즈니스/규제) 4. 중요도 태그 (상/중/하) 자동 분류 5. 결과를 표 형태로 정리 + CSV 다운로드 디자인: 깔끔하고 전문적인 스타일 기본 검색어: 'LG AI 전환', '산업별 AI 전략 2026', 'AI 경쟁 동향'

**Phase 1 완료 후:** 기본적인 리서치 대시보드가 동작합니다

Phase 2 — 기능 고도화 (20분)

Phase 1에서 만든 앱에 **같은 대화에서 계속** 기능을 추가합니다. 아래 프롬프트를 하나씩 입력하세요.

2-1 감성 분석 추가복사

각 뉴스 기사의 요약 옆에 감성 분석 태그를 추가해줘. - 긍정(🟢) / 부정(🔴) / 중립(⚪) 3단계 - 감성 기준으로 필터링할 수 있는 버튼도 추가

2-2 KPI 대시보드 추가복사

대시보드 상단에 KPI 카드 영역을 추가해줘: - 총 기사 수 - 긍정/부정 비율 (파이 차트) - 가장 많이 언급된 키워드 Top 5 - 최신 업데이트 시간

2-3 날짜 정렬 & 북마크복사

다음 기능을 추가해줘: 1. 검색 결과를 날짜순/중요도순으로 정렬하는 버튼 2. 중요한 기사에 ⭐ 북마크 기능 (클릭하면 토글) 3. 북마크한 기사만 모아보는 탭

**Phase 2 완료 후:** 감성 분석, KPI, 정렬, 북마크가 추가된 고도화된 대시보드

10분 휴식

Phase 2까지 완료한 후, 다음 단계로 넘어가기 전에 **10분간 휴식**을 가집니다.

Phase 3 — LG 브랜드 UI/UX (20분)

기능이 완성된 앱의 **외관을 LG 브랜드에 맞게 전문적으로 다듬습니다.**

3-1 LG 브랜드 컬러 적용복사

디자인을 LG 브랜드에 맞게 전면 수정해줘: - 메인 컬러: #A50034 (LG 매젠타) - 서브 컬러: #6B6B6B (그레이) - 배경: 깔끔한 화이트 기반 - 폰트: 시스템 산세리프 (Apple SD Gothic Neo 등) - 좌측 상단에 "AX Intelligence Hub" 타이틀 표시 - 전체적으로 경영진 보고에 적합한 격식 있는 디자인

3-2 다크모드 & 반응형복사

다음을 추가해줘: 1. 우측 상단에 다크모드 토글 버튼 (🌙/☀️ 아이콘) 2. 다크모드에서도 LG 매젠타 포인트 유지 3. 모바일에서도 깔끔하게 보이도록 반응형 디자인 적용 4. 테이블이 작은 화면에서 가로 스크롤 가능하도록

3-3 마무리 터치복사

마지막으로 다음을 다듬어줘: 1. 로딩 시 스켈레톤 UI 적용 (회색 박스 → 실제 데이터) 2. 검색 버튼에 호버 애니메이션 3. KPI 카드에 아이콘과 미세한 그림자 효과 4. 푸터에 "Powered by Google AI Studio · LG AX Camp" 표시

**Phase 3 완료 후:** LG 브랜드 아이덴티티가 반영된 프로페셔널 대시보드 완성!

Phase 4 — 상급 확장 선택 (15분)

시간이 남거나 더 도전하고 싶은 분을 위한 상급 과정입니다.

\[상급\] Firecrawl API**API**  
프로그램끼리 대화하는 통로예요. 마치 레스토랑의 웨이터처럼, 주문(요청)을 주방(서비스)에 전달하고 요리(결과)를 가져다줍니다. 연동복사

추가 기능: Firecrawl API를 연동하여 특정 웹사이트의 콘텐츠를 직접 크롤링하는 기능을 추가해줘. Firecrawl 연동: - API Key: (본인의 Firecrawl API Key 입력) - 크롤링 대상: 경쟁사 IR 페이지, 뉴스룸, 블로그 - 크롤링 결과를 마크다운으로 변환하여 대시보드에 통합 - 정기 크롤링 스케줄 설정 UI 추가

자유 확장 아이디어

• "검색 히스토리를 저장하고 이전 검색을 불러오는 기능 추가해줘"  
• "주간 리포트를 자동 생성하여 이메일로 보내는 기능 추가해줘"  
• "경쟁사별 뉴스 빈도 추이를 라인 차트로 시각화해줘"  
원하는 기능을 자유롭게 말로 설명하면 AI가 구현합니다!
