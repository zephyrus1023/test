HTML + PPT 동시 출력 심화

이 섹션은 심화/선택 자료입니다. 시간 여유가 있을 때 참고하세요.
앞서 학습한 듀얼 포맷 개념을 바탕으로, 분석 결과를 인터랙티브 HTML 대시보드 + 이사회 보고용 PPT로 직접 실습합니다.
이 실습에서는
• 같은 분석 결과를 HTML 대시보드 + PPT 두 가지 포맷으로 동시에 출력합니다
• 실무 공유용(HTML)과 이사회 보고용(PPT)의 용도 차이를 체험합니다
• LG 브랜드 컬러가 적용된 실전 수준의 산출물을 만듭니다
Step 3: HTML + PPT 동시 출력 프롬프트

이어서 같은 대화에서 입력합니다 (새 대화를 열지 마세요):

HTML + PPT 동시 생성
앞서 분석한 내용을 기반으로 두 가지 형태의 산출물을 동시에 생성해주세요. [산출물 1: HTML 대시보드] - 인터랙티브 KPI 카드 (매출, 점유율, 고객만족도, AI 탑재율) - 차트 3종: 매출 추이(Line), 부문별 비중(Donut), 경쟁사 비교(Bar) - 필터링/정렬 가능한 데이터 테이블 - 반응형 디자인, 다크/라이트 모드 지원 - LG 브랜드 컬러: Heritage Red #A50034 [산출물 2: PowerPoint] - 16:9 와이드, 5~7 슬라이드 - LG 브랜드 컬러: Heritage Red #A50034 - 구성: 표지 → Executive Summary → 매출 실적 → 전략 과제 → Next Steps - 이사회 보고 수준의 간결한 레이아웃 - 슬라이드당 핵심 메시지 1개 원칙
산출물 확인 포인트
HTML 대시보드 체크
  • KPI 카드 4개 표시
  • Heritage Red(#A50034) 적용
  • 차트 3개 정상 렌더링
  • 필터링/정렬 동작
  • 다크/라이트 모드 전환
  • 모바일 반응형
PPT 체크
  • 5~7 슬라이드 구성
  • Heritage Red 브랜드 컬러
  • Executive Summary 포함
  • 차트/그래프 포함
  • 여백 충분 (임원 보고 적합)
  • 16:9 와이드스크린
