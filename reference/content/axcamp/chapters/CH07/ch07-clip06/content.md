# 스킬(Skill)이란 심화

이 섹션은 **심화/선택** 자료입니다. 시간 여유가 있을 때 참고하세요.

반복 업무를 자동화하는 **스킬(Skill)**의 개념과 원리를 이해하고, 조직 확산 시나리오를 살펴봅니다.

스킬(Skill)이란?

Antigravity**Antigravity**  
AI 에이전트 플랫폼이에요. 에디터·터미널·브라우저를 AI가 동시에 조작하여 코드 작성, 파일 생성, 웹 작업을 한꺼번에 자율적으로 처리합니다.의 **스킬**은 자주 사용하는 프롬프트 + 워크플로우를 저장해두고 클릭 한 번으로 재실행하는 기능입니다.

스킬은 **조직의 업무 표준(SOP)을 디지털화**한 것입니다. 한 명이 만들면 팀 전체가 같은 품질로 산출물을 만들 수 있습니다.

스킬의 조직 확산 시나리오

1명이 만들면

전무가 최적의 프롬프트를  
스킬로 저장

팀 전체가 쓰고

팀원 6명이 동일한  
품질의 산출물 생성

조직 표준이 된다

사업부 전체의  
보고 프로세스 통일

이것이 **"AI 시대의 SOP"**입니다. 개인의 노하우가 조직의 자산으로 전환됩니다.

스킬 활용의 핵심 — "한 번 만들면 계속 쓴다"

이번 실습에서 만든 스킬은 실제 업무에서 바로 활용 가능합니다:  
  
• **매주 월요일**: 주간 보고서 → 스킬 실행 → 3종 보고서 자동 생성  
• **매월 초**: 월간 실적 → 스킬 실행 → 대시보드 + PPT 자동 생성  
• **분기말**: 분기 전략보고서 → 스킬 실행 → 이사회 자료 초안 5분 완성  
  
반복 업무에 들이는 시간을 **90% 이상** 절약할 수 있습니다.
