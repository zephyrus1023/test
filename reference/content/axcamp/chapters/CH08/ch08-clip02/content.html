Copilot 참고자료

Microsoft Copilot의 핵심 기능과 기업 환경에서의 활용법을 정리한 참고 자료입니다.
Microsoft Copilot이란?
  • Microsoft 365 통합: Word, Excel, PowerPoint, Outlook, Teams에 AI가 내장
  • 업무 자동화: 이메일 요약, 문서 초안, 프레젠테이션 생성, 데이터 분석을 각 앱 내에서 처리
  • 기업 보안: Microsoft 365 Enterprise의 보안 체계 내에서 동작, 데이터가 외부로 유출되지 않음
  • Graph 연동: 사내 이메일, 문서, 캘린더, 채팅 기록 등을 통합하여 맥락 있는 답변 제공
