Hi-D Code 소개 및 시연

최남석(Agentic AI 팀)님이 자사 Vibe Coding 환경인 Hi-D Code의 구축 현황과, 이미지상 불량 검출 S/W 개발 사례를 중심으로 활용 가능성을 소개합니다.
무엇을 보나
  • Hi-D Code 를 소개합니다
  • 자사의 Vibe Coding 환경 (Hi-D Code)의 구축 현황
  • Hi-D Code 환경을 사용한 간단한 App 개발 사례 소개: 이미지상 불량 검출 S/W 개발
짧은 시연 흐름
단계 설명
1. 목표입력 과제 프롬프팅시 어떤 목적의 App 을 만드는지, 제반사항은 무엇인지 설명합니다. (e.g. 불량검출 App)
2. 실행확인 AI 가 작업을 어떻게 수행하는지 확인하고 필요시 코드 수정을 지시합니다.
3. 결과검토 생산된 산출물을 보고 수정 방향을 다시 바로잡습니다.
