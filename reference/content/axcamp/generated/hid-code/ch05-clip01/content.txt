Hi-D Code 소개 및 시연

최남석(Agentic AI 팀)님이 자사 Vibe Coding 환경인 Hi-D Code의 구축 현황과, 이미지상 불량 검출 S/W 개발 사례를 중심으로 활용 가능성을 소개합니다.

무엇을 보나

- Hi-D Code 를 소개합니다

- 자사의 Vibe Coding 환경 (Hi-D Code)의 구축 현황

- Hi-D Code 환경을 사용한 간단한 App 개발 사례 소개: 이미지상 불량 검출 S/W 개발
