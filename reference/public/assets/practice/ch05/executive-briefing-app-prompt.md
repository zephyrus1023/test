# Executive Briefing Formatter Prompt

경영진 보고용 초안을 빠르게 정리하는 React 웹앱을 만들어줘.

핵심 기능:
- 긴 메모나 회의 내용을 붙여넣으면
- 1페이지 브리핑 형식으로 요약한다
- 아래 4개 블록으로 출력한다
  - Situation
  - Key Signals
  - Decision Points
  - Recommended Actions

UI 요구사항:
- 임원 보고 문서처럼 차분하고 정돈된 레이아웃
- 카드형 섹션
- 복사 버튼과 Markdown 다운로드 버튼 제공

스타일:
- 화이트 배경
- 짙은 네이비 텍스트
- 액센트 컬러는 #A50034

출력 톤:
- 간결하고 격식 있는 비즈니스 문체
