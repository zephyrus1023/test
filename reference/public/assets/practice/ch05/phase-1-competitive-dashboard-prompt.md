# Phase 1: Competitive Dashboard Prompt

경쟁사의 최신 뉴스와 기술 동향을 자동으로 수집·요약하는 리서치 대시보드를 만들어줘.

기능 요구사항:
1. 검색어 입력
2. Google Search 기반 관련 뉴스 검색
3. 각 결과를 3줄로 요약
4. 카테고리 필터: 기술 / 비즈니스 / 규제
5. 중요도 태그: 상 / 중 / 하
6. 표 형태 정리 + CSV 다운로드

기본 검색어:
- LG AI 전환
- 산업별 AI 전략 2026
- AI 경쟁 동향

스타일:
- 경영진이 보기 좋은 깔끔한 대시보드
- 화이트 배경
- 선명한 검색 영역
- 카드형 KPI 영역이 추가되기 쉬운 구조
