# Phase 2: Refinement Prompts

## 2-1 감성 분석 추가

각 뉴스 기사의 요약 옆에 감성 분석 태그를 추가해줘.
- 긍정(🟢) / 부정(🔴) / 중립(⚪) 3단계
- 감성 기준 필터 버튼도 추가

## 2-2 KPI 카드 추가

대시보드 상단에 KPI 카드 영역을 추가해줘.
- 총 기사 수
- 긍정/부정 비율
- 가장 많이 언급된 키워드 Top 5
- 최신 업데이트 시간

## 2-3 정렬과 북마크

다음 기능을 추가해줘.
1. 날짜순 / 중요도순 정렬
2. 기사 북마크 토글
3. 북마크한 기사만 보는 탭
