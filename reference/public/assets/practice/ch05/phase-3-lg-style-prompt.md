# Phase 3: LG Style Prompt

디자인을 LG 스타일의 경영진 보고 대시보드로 다듬어줘.

브랜드 가이드:
- 메인 컬러: #A50034
- 보조 컬러: #6B6B6B
- 배경: 화이트 기반
- 제목: AX Intelligence Hub

추가 요구사항:
1. KPI 카드는 정리된 카드형으로 보여줘
2. 검색 버튼에 미세한 hover 효과를 넣어줘
3. 로딩 시 스켈레톤 UI를 적용해줘
4. 모바일에서도 흐트러지지 않게 반응형으로 구성해줘
5. 다크모드 토글을 우측 상단에 추가해줘

톤:
- 화려하기보다 신뢰감 있는 기업형 UI
- 경영진 보고에 적합한 절제된 인터페이스
