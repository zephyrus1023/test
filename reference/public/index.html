Root Editor

현재 클립 편집

root 전용 편집기입니다. 저장하면 현재 클립의 content.html, content.md, metadata.json, content.txt가 함께 동기화됩니다. 저장 전 원본은 자동 백업되며, 아래에서 현재 클립용 자산도 바로 업로드할 수 있습니다.

Clip Assets

이미지 · PDF · 오디오 · 동영상 자산

업로드한 파일은 현재 클립 폴더의 assets/ 아래 저장됩니다. 경로 복사, HTML 삽입, 미리보기를 바로 사용할 수 있습니다.
